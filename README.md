# ecohub-webapp
Ecohub Web Application using Spring Boot


## Build and run

* Clean and package jar file
```
./mvnw clean package 
```

* Run webapp
```
./run.sh
```

* Run in development mode
```
./mvnw spring-boot:run
```

* Open application in default browser
```
open http://localhost:9090/ecohub
```
